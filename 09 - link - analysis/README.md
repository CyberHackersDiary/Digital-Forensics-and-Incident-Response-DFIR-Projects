# 🔗 Link Analysis

> Identifying, mapping, and analysing relationships between people, organisations, domains, and digital entities through open-source intelligence and evidence-based investigation — from isolated data points to a fully visualised intelligence network.

Every investigation in this folder starts with a set of entities or indicators of interest and ends with a documented, reproducible network intelligence assessment. No assumptions. No shortcuts. Evidence in, intelligence out.

---

## 🎯 What This Covers

| Technique | What It Involves |
|---|---|
| **Relationship Mapping** | Identifying and charting connections between people, organisations, and digital entities |
| **Entity Analysis** | Examining individual nodes — people, companies, domains, accounts — to build intelligence profiles |
| **Network Visualisation** | Representing complex relationship data as clear, structured network graphs |
| **Intelligence Correlation** | Cross-referencing entity data across multiple sources to surface hidden connections |
| **Timeline Analysis** | Reconstructing the chronological sequence of events and interactions within a network |
| **Association Discovery** | Uncovering indirect links and non-obvious relationships between entities |
| **Pattern Recognition** | Identifying recurring behaviours, structural patterns, and anomalies within networks |
| **Network Investigation** | Conducting structured investigations into the structure and activity of digital networks |
| **Data Visualisation** | Transforming raw intelligence data into clear, interpretable visual network representations |
| **Evidence Documentation** | Recording all findings in structured, verifiable, and reproducible investigation reports |

## 🔬 Investigative Methodology

Each investigation follows a disciplined, repeatable eight-stage workflow:

```
╔══════════════════════════════════════════════════════════════╗
║               INVESTIGATIVE METHODOLOGY                      ║
╠══════════════════════════════════════════════════════════════╣
║                                                              ║
║  01  RESEARCH QUESTION                                       ║
║      Define the entities of interest and the relationships   ║
║      to be investigated                                      ║
║                              ↓                               ║
║  02  COLLECTION                                              ║
║      Gather publicly available data on all identified        ║
║      entities, associations, and digital indicators          ║
║                              ↓                               ║
║  03  VERIFICATION                                            ║
║      Validate source credibility and entity data integrity   ║
║                              ↓                               ║
║  04  CORRELATION                                             ║
║      Cross-reference entity relationships across multiple    ║
║      platforms and independent intelligence sources          ║
║                              ↓                               ║
║  05  ANALYSIS                                                ║
║      Apply link analysis techniques to map networks,         ║
║      identify patterns, and surface hidden associations      ║
║                              ↓                               ║
║  06  FINDINGS                                                ║
║      Establish key conclusions on entity relationships,      ║
║      network structure, and intelligence significance        ║
║                              ↓                               ║
║  07  INTELLIGENCE ASSESSMENT                                 ║
║      Assess confidence level, network implications,          ║
║      and alternative relationship interpretations            ║
║                              ↓                               ║
║  08  REPORT                                                  ║
║      Document conclusions in a structured, reproducible      ║
║      link analysis intelligence report                       ║
║                                                              ║
╚══════════════════════════════════════════════════════════════╝
```

Every conclusion is evidence-backed. Every step is reproducible.

## ⚠️ Responsible Use

All investigations in this folder are conducted strictly using open-source intelligence and publicly available data:

- **Legal compliance** — All methods comply with applicable laws and platform terms of service
- **Public data only** — No unauthorised access to private, restricted, or confidential information
- **Privacy by design** — Entity mapping is limited to what is necessary and proportionate to the investigation
- **Responsible handling** — Sensitive network findings and entity associations are handled with appropriate care
- **Educational purpose** — All case studies are conducted for learning and portfolio demonstration only
