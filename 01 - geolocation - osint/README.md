# 🗺️ Geolocation Investigations

> Pinpointing the location of images and videos using visual clues, satellite imagery, open-source mapping, and systematic evidence-based analysis — from a single photograph to a verified coordinate.

Every investigation in this folder starts with raw, unverified visual evidence and ends with a documented, reproducible location finding. No guesswork. No assumptions. Evidence in, intelligence out.

---

## 🎯 What This Covers

| Technique | What It Involves |
|---|---|
| **Landmark Identification** | Recognising monuments, structures, and geographic features unique to a region |
| **Road Sign Analysis** | Extracting location data from signage, lane markings, and road infrastructure |
| **Terrain & Environment** | Reading landscape, vegetation, elevation, and topography as geolocation indicators |
| **Architecture & Infrastructure** | Identifying regional building styles, utility configurations, and construction patterns |
| **Shadow & Sun Position Analysis** | Estimating location and time-of-day using shadow direction and solar angle calculations |
| **Satellite Imagery Comparison** | Cross-referencing ground-level visuals against overhead and aerial imagery |
| **Street View Verification** | Confirming candidate locations through panoramic ground-level mapping platforms |
| **Image-to-Map Correlation** | Precisely matching visual elements and perspective to geographic coordinates |
| **Video Geolocation** | Extracting key frames from footage and applying geolocation techniques to moving visuals |
| **Evidence Documentation** | Recording all findings in structured, verifiable, and reproducible investigation reports |

## 🔬 Investigative Methodology

Each investigation follows a disciplined, repeatable eight-stage workflow:

```
╔══════════════════════════════════════════════════════════════╗
║               INVESTIGATIVE METHODOLOGY                      ║
╠══════════════════════════════════════════════════════════════╣
║                                                              ║
║  01  RESEARCH QUESTION                                       ║
║      Define what needs to be located and why                 ║
║                              ↓                               ║
║  02  COLLECTION                                              ║
║      Gather all available visual and open-source evidence    ║
║                              ↓                               ║
║  03  VERIFICATION                                            ║
║      Validate evidence integrity and source credibility      ║
║                              ↓                               ║
║  04  CORRELATION                                             ║
║      Cross-reference clues across multiple sources           ║
║                              ↓                               ║
║  05  ANALYSIS                                                ║
║      Apply geolocation techniques to extract indicators      ║
║                              ↓                               ║
║  06  FINDINGS                                                ║
║      Establish the most probable location with evidence      ║
║                              ↓                               ║
║  07  INTELLIGENCE ASSESSMENT                                 ║
║      Assess confidence level and alternative hypotheses      ║
║                              ↓                               ║
║  08  REPORT                                                  ║
║      Document conclusions in a reproducible format           ║
║                                                              ║
╚══════════════════════════════════════════════════════════════╝

```

Every conclusion is evidence-backed. Every step is reproducible.
