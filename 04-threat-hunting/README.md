# Threat Hunting
Proactive, hypothesis-driven search for threats that have evaded automated defences. Uses threat intelligence, adversary behaviour models, and telemetry analysis to detect stealthy attackers and translate findings into new detections and hardening measures.

## Skills Demonstrated
* Hypothesis Development & Testing
* MITRE ATT&CK Mapping & Application
* SIEM Query Development & Log Analysis
* EDR Telemetry Analysis
* Behavioural Anomaly Detection
* Living-off-the-Land (LotL) Technique Detection
* Lateral Movement & Privilege Escalation Identification
* Persistence Mechanism Discovery
* Command & Control (C2) Identification
* Threat Intelligence Integration
* IOC & IOA Development
* User & Entity Behaviour Analytics (UEBA)
* Hunt Playbook Development
* Detection Engineering & Rule Creation
* Data Stacking & Frequency Analysis
* Campaign & Actor Attribution
