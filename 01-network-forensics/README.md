# Network Forensics
Analysis of network traffic and events to identify malicious activity, reconstruct attacks, and support incident response. Covers packet-level inspection through application-layer protocols across both real-time and historical data.

## Skills Demonstrated
* Packet Capture (PCAP) Analysis
* Protocol Analysis & Decoding (HTTP, DNS, FTP, SMB, TLS, ICMP)
* Traffic Baseline & Anomaly Detection
* DNS Investigation & Tunnelling Detection
* Command & Control (C2) Traffic Identification
* Lateral Movement Detection
* Network Intrusion Detection & Signature Analysis
* Flow Analysis (NetFlow / IPFIX / sFlow)
* Firewall & Proxy Log Analysis
* Wireless Network Forensics (802.11)
* VPN & Encrypted Traffic Analysis
* Session Reassembly & Traffic Reconstruction
* Exfiltration Detection
* IOC Extraction & Correlation
* Threat Intelligence Integration
* Evidence Preservation & Chain of Custody
